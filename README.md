# Bitcoin Prediction Dashboard

This project predicts Bitcoin price using historical data and displays results in a Streamlit dashboard.

## Features
- Live Bitcoin price data (Binance API)
- Prediction range using statistical simulation
- Interactive dashboard

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy

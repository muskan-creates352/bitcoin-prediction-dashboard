# 🚀 Bitcoin Prediction Dashboard

![Dashboard Screenshot](https://raw.githubusercontent.com/muskan-creates352/bitcoin-prediction-dashboard/main/Screenshot%202026-04-30%20121647.png)

## 📌 Overview
This project is a Bitcoin price prediction dashboard built using Streamlit.  
It fetches real-time data from Binance API and predicts future price range using statistical simulation.

---

## ✨ Features
- 📊 Live Bitcoin price data (Binance API)
- 📈 Prediction range using Monte Carlo simulation
- 📉 Interactive price chart
- ⚡ Fast and simple UI with Streamlit

---

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Requests API
